# 2024ChungKang_GameProject_RoguePlatforemer
청강대 2024년 2학기 게임프로젝트 입문 팀프로젝트 리포지터리



---



### 폴더 구조
* **01.Senes** : *게임의 각 씬*
* **02.Scripts** : *스크립트(코딩)* 
* **03.Prefabs** : *프리펩*
* **04.Resources** : *게임에 사용되는 리소스(이미지,폰트...)*
* **05.Materials** : *게임 개발에 사용되는 메테리얼(재질)*
* **06.Sounds** : *사운드*
* **07.Animations** : *게임 캐릭터 애니메이션*
* **08.Effects** : *파티클 & 이펙트*
